1. Informações do Processo

- Cliente
- Nª do processo
- Área de atuação
- Situação (InitialStage = 0)
- Prazo Final
- Pro bono? (Se sim, exibir informações do valor)

2. Informações sobre o pagamento

- Valor Total
- Quantidade de parcelas
- Emitir Nota
- Link para anexos do contrato (Opcional)
